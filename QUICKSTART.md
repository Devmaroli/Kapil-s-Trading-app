# Quick Start Guide

Get the trading app up and running in 5 minutes!

## One-Line Setup

```bash
git clone https://github.com/Devmaroli/Kapil-s-Trading-app.git && cd Kapil-s-Trading-app && npm install && cp .env.example .env && npm start
```

## Step-by-Step Setup

### Step 1: Clone the Repository
```bash
git clone https://github.com/Devmaroli/Kapil-s-Trading-app.git
cd Kapil-s-Trading-app
```

### Step 2: Install Dependencies
```bash
npm install
```

### Step 3: Configure Environment
```bash
cp .env.example .env
# Open .env and update with your settings
```

### Step 4: Run the App
```bash
npm start
```

Open your browser and go to `http://localhost:3000` 🚀

## Troubleshooting

### Port 3000 already in use?
```bash
PORT=3001 npm start
```

### Dependencies not installing?
```bash
rm -rf node_modules package-lock.json
npm install
```
